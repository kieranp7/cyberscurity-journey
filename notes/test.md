## CompTIA A+ — Modules 4–7 notes

### Module 4: Operating Systems
- OS installation types: clean install, upgrade, repair, multiboot
- Storage partitioning: MBR vs GPT, primary/extended/logical partitions
- Windows editions and upgrade paths
- Command line basics: ipconfig, chkdsk, sfc, diskpart
- Linux fundamentals: directory structure, package managers, basic commands

### Module 5: Virtualisation and Cloud Computing
- Hypervisor types: Type 1 (bare metal) vs Type 2 (hosted)
- Cloud service models: IaaS, PaaS, SaaS
- Cloud deployment models: public, private, hybrid, community
- Why virtualisation matters for resource efficiency and isolation

### Module 6: Security
- Physical vs logical security controls
- Windows security: NTFS vs share permissions, Active Directory basics
- Malware types: virus, worm, trojan, ransomware, spyware
- Social engineering techniques: phishing, pretexting, tailgating
- Data destruction: wiping vs degaussing vs physical destruction

### Module 7: Operational Procedures
- Change management process and documentation
- Incident response basics and chain of custody
- Safety procedures and ESD precautions
- Backup types: full, incremental, differential

### Key takeaway
Modules 4–7 tie together the practical side of IT support — from OS administration through to the security and procedural knowledge needed to operate safely in a live environment. Internal assessment passed with this material.
